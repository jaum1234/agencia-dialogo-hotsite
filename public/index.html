PROJETO

A Nike fechou uma parceria com o jogador de basquete norte-americano Kevin Durant, em que haverão algumas peças da marca autografadas por ele! Foram distribuídos vários QR Codes por diversas quadras de São Paulo, em que alguns serão premiados!

A iniciativa do projeto tem como base o incentivo à prática de esportes, voltado ao basquete, além de reforçar o uso de quadras públicas espalhadas pela cidade de São Paulo.

COMO PARTICIPAR ?

  1. Visite as quadras de São Paulo e procure por algum QR Code
  2. Aponte a câmera para o QR Code para escaneá-lo
  3. Uma página na web será aberta, preencha com seus dados
  4. Após isso, torça para ser o premiado!
